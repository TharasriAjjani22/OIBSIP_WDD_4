# OIBSIP_WDD_4
Instagram Login & Feed UI Clone
📌 Objective:
The objective of this project is to build a simple Instagram-inspired login and feed interface. This project focuses on user authentication concepts and creating visually appealing UI using modern frontend tools.

🚀 What I Did (Steps Performed):
Created the login/signup interface using HTML
Designed a modern UI inspired by Instagram using Tailwind CSS
Implemented authentication logic using JavaScript:
Login functionality
Signup (registration)
Basic validation
Used localStorage to store user credentials
Built a simple feed page UI that appears after successful login
Added features like:
Toggle between login and signup
Logout functionality

🛠️ Tools & Technologies Used
HTML5 – Structure of the application
Tailwind CSS – Styling and responsive design
JavaScript (Vanilla JS) – Logic and interactivity
LocalStorage – Storing user data
VS Code – Development environment
Git & GitHub – Version control and hosting

💡 Features
🔑 User login and signup system
💾 Stores user credentials using localStorage
🔄 Toggle between Login and Sign Up
⚠️ Basic form validation
🏠 Instagram-like feed UI after login
🚪 Logout functionality
🎨 Modern and responsive design

📊 Outcome
This project helped me understand:
Basics of authentication flow
How localStorage works in web applications
Handling user input and validation
Creating real-world UI designs using Tailwind CSS
It improved both my frontend design skills and JavaScript logic building.

Final Thoughts
This project is a beginner-friendly simulation of how login systems work. While it is not a secure real-world authentication system, it helped me understand the core concepts.
In the future, I plan to improve this by:
Connecting it with a backend (Node.js & database)
Adding real authentication (JWT, sessions)
Enhancing UI with animations
